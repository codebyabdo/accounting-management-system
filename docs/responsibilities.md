- Team Lead
- 2 Front-End Developers
- 1 Back-End Developer

## Team Lead

Responsible for:

- Project Planning
- Architecture
- Code Review
- Task Distribution
- Final Deployment

- UI Development
- Components

- Responsive Design
- Dark Mode
- Translation

## Back-End Developer

Responsible for:

- API Development
- Authentication
- Database
- Business Logic
- Security

- Pull Requests
